## Examples

基本按钮样式:

```jsx
<example-button>Primary</example-button>
<example-button type="secondary">Secondary</example-button>
<example-button type="success">Success</example-button>
<example-button type="danger">Danger</example-button>
<example-button type="warning">Warning</example-button>
<example-button type="info">Info</example-button>
<example-button type="light">Light</example-button>
<example-button type="dark">Dark</example-button>
<example-button type="link">Link</example-button>
```

Outline按钮样式:

```jsx
<example-button outline>Primary</example-button>
<example-button type="secondary" outline>Secondary</example-button>
<example-button type="success" outline>Success</example-button>
<example-button type="danger" outline>Danger</example-button>
<example-button type="warning" outline>Warning</example-button>
<example-button type="info" outline>Info</example-button>
<example-button type="light" outline>Light</example-button>
<example-button type="dark" outline>Dark</example-button>
<example-button type="link" outline>Link</example-button>
```

按钮大小:

```jsx
<example-button size="lg">Large</example-button>
<example-button type="secondary" size="lg">Large</example-button>
<example-button size="sm">Small</example-button>
<example-button type="secondary" size="sm">Small</example-button>
```

块状按钮:

```jsx
<example-button block>Block</example-button>
<example-button type="secondary" block>Block</example-button>
```

禁用状态:

```jsx
<example-button disabled>Primary</example-button>
<example-button type="secondary" disabled>Secondary</example-button>
<example-button type="success" disabled>Success</example-button>
<example-button type="danger" disabled>Danger</example-button>
<example-button type="warning" disabled>Warning</example-button>
<example-button type="info" disabled>Info</example-button>
<example-button type="light" disabled>Light</example-button>
<example-button type="dark" disabled>Dark</example-button>
<example-button type="link" disabled>Link</example-button>
```
